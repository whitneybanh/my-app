■ご利用について
TrueTypeのフリーフォントです。 
個人・商用問わず無料でお使いいただけます。
印刷物・イラスト・バナー・映像・ゲーム・同人系等、何でもどうぞ。

基本的に使用連絡は不要です。
商標登録する物に使用する場合は必ず事前連絡お願いいたします。


■フォントについて
MG01_bmp8
欧文フォント。8の倍数の場合、綺麗に表示されます。
英字・半角英数字・記号
武蔵システム様（http://musashi.or.tv/）のソフトTTEditにて作成しています。


■注意・禁止事項
フォントデータの改造・販売・無断での再配布は禁止します。
掲載フォントの著作権はtogaにあります。
予告無しに配布終了・改訂する場合がございます。



■インストール
解凍したmg01_bmp8.ttfをフォントフォルダに入れてください。
よくわからない方は「フォント インストール」等で検索してみてください。

TrueTypeフォント対応のOSでしたら、多分動作可能だと思います。
